FEATURES

  • Low current (max. 200 mA)

  • Low voltage (max. 25 V).


    APPLICATIONS

  • General purpose switching and amplification,

e.g. small-signal audio-frequency applications.


DESCRIPTION

NPN transistor in a TO-92; SOT54 plastic package. PNP complement: 2N4126.
